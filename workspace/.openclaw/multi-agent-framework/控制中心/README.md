# OpenClaw 多智能体控制中心

多 AI 智能体协作框架，7×24 小时无休、分工明确的 AI 团队

## 快速启动

```bash
# 启动控制中心
python control-center.py

# 或双击运行
start-control-center.bat
```

## 团队架构

| 智能体 | 形象 | 角色 | 状态 |
|--------|------|------|------|

## 配置文件

- `team-config.json` - 团队配置
- `agents/` - 各智能体配置
  - `may/` - 核心主控
  - `gock/` - 每日新闻
  - `otter/` - 私人助理
  - `pandas/` - 开发测试
  - `monkey/` - 内容创作
  - `tiger/` - 安全更新
