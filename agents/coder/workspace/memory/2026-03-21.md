# 2026-03-21 工作日志

## 军机处 Dashboard 故障排查

### 问题描述
- Dashboard (http://127.0.0.1:7891/) 显示"同步异常"，旨意数据为 0
- 页面显示"服务器未启动"

### 排查结果
1. **7891 端口服务未运行** - Python 进程被终止
   - 解决：重启了 `server_win.py`

2. **各省 Agent 无 sessions.json** - shangshu、menxia、hubu、libu、bingbu、xingbu、gongbu、libu_hr、zaochao、zhongshu 都没有生成会话文件
   - 原因：各省 Agent 从未被调用运行过
   - 解决：创建了空的 sessions.json 文件

3. **任务数据为空**
   - 原因：各省 Agent 没有运行，没有产生任务
   - 临时解决：手动在 tasks_source.json 添加了 2 个测试任务

- 现在省部调度页面显示每个部门的职责（如：尚书省"总辖六部、政策执行、监督落实"）
