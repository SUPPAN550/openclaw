## buildwithclaude 仓库分析

从 https://github.com/davepoon/buildwithclaude.git 下载并分析了该仓库：

### 仓库内容
- **plugins/all-agents/**: 150+ 个专业 Agent 配置
- **plugins/all-commands/**: 200+ 个命令（代码审查、部署、测试等）
- **plugins/all-skills/**: 100+ 个技能（自动化集成、文档处理等）
- **plugins/all-hooks/**: 30+ 个钩子（Git、通知、安全等）
- **web-ui/**: Next.js 构建的 Web 界面

### 启动的 Web UI
- 地址: http://localhost:3000
- 功能: 浏览 Agents、Commands、Hooks、MCP 服务器等
- 技术栈: Next.js 16 + React 19 + Tailwind CSS 4

### MCP 服务器说明
- 功能: 让 Claude 调用外部 API 和工具
- 示例: 房贷计算器、数据库查询、Git 操作等
- 配置方式: JSON 格式定义服务器连接
