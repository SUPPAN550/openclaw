# 2026-03-20 工作日志

## Paperclip AI 部署

- 从 GitHub 部署 Paperclip AI 项目：https://github.com/paperclipai/paperclip
- 安装并配置 PostgreSQL 17 数据库
- 通过 npm 全局安装 Claude Code (`npm install -g @anthropic-ai/claude-code`)
- 创建数据库用户 paperclip 和数据库 paperclip
- 配置环境变量：DATABASE_URL, SERVE_UI=true
- 启动服务器在 http://127.0.0.1:3100
