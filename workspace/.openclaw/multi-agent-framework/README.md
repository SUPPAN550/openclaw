## 三层架构

### 1. 全局层 (Global)
- 大模型厂商 / 默认模型
- Agent 列表、工作目录
- 全局工具权限
- Telegram/Discord 连接
- 全局记忆配置

### 2. Agent 层 (Agent)
- 工作说明书 (SOUL.md)
- 性格风格
- 用户理解
- 身份卡 (IDENTITY.md)
- 工具说明书 (TOOLS.md)
- 长期精华记忆 (MEMORY.md)
- 专属技能

### 3. Session 层 (Session)
- 独立对话场景
- 不同渠道隔离
- 并行对话支持

## 协作模式

### 固定分工协作
```
Gock(情报) → May(筛选) → Monkey(转文章) → Pandas(优化)
```

### 临时子代理
- 主 Agent 拉起独立子 Agent
- 隔离 Session 执行重任务
- 不阻塞主对话
